# Pangyo Startup Campus Bldg 1, 7F — Digital Twin Viewer

Interactive web viewer for the 7F Gazebo simulation world (rev 4fe6291).

- **View online:** open the GitHub Pages URL of this repo
- `index.html` — fully self-contained viewer (three.js + model inlined)
- `floor7f.glb` — the model in standard glTF binary, usable in any 3D viewer

Generated from the Gazebo SDF world by a Blender headless conversion pipeline.
